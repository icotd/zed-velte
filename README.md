# Zed Velte

A [Velte](https://velte.dev) extension for [Zed](https://zed.dev).

This build is intentionally lightweight and provides grammar-based syntax
highlighting/editing support for `.velte` files without bundling language
server integrations.

## Development

To develop this extension, see the [Developing Extensions](https://zed.dev/docs/extensions/developing-extensions) section of the Zed docs.
